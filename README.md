Project name : Digitools.

Description : Digitools is a modern SaaS product marketplace UI built with React.
It allows users to browse products, add them to a cart,
and manage selections with real-time feedback. The app features dynamic data loading,
local storage persistence, 
toast notifications, and a responsive design for a smooth user experience.

Technology: React.js,tailwind css, daisyUI,

Features: 
1. Add to Cart System
2. Dynamic Button State
3. Remove from Cart
4. Local Storage Integration
5. Toast Notifications
